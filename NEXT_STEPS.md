## Quick Start Commands

### Test Everything
```bash
# Run unit tests
npm run test:unit

# Run with coverage
npm run test:coverage

# Compile TypeScript
npm run compile

# Lint code
npm run lint
```

### Package Extension
```bash
# Package with git tagging
npm run package

# Package for development (no git tag)
npm run package:dev

# Test locally
code --install-extension code-context-notes-0.1.0.vsix
```

### Publish Extension
```bash
# Publish to both marketplaces
npm run publish

# Or use bash script
npm run publish:bash
```
